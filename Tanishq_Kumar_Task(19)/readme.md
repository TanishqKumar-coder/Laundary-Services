# Laundry Service Booking Website
A responsive frontend project for a Laundry Service Booking platform built using HTML, CSS, and JavaScript.

This project allows users to browse laundry services, add them to a cart, view the total amount, and book services through a form with proper validation.
---

## Features
- Responsive UI Design
- Dynamic Service Browsing
- Add to Cart Functionality
- Automatic Total Price Calculation
- Booking Form Validation
- Error and Success Messages
- Modern Card-Based Layout
- Interactive Buttons and Navbar
---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome
